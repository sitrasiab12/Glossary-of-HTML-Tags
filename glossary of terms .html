Basic HTML Tags

some tags of html

  • <html> : Root element
  • <head> : Metadata container
  • <body> : Visible content
  • <p> : Paragraph
  • <a> : Hyperlink

let's see them one by one

<html>

The root element of an HTML document. All content is placed inside this tag.

<head>

Contains meta information about the webpage like title, styles, and links.

<body>

Contains all visible content such as text, images, links, and sections.

<h1> to <h6>

Heading tags used to define titles and subtitles in different sizes.

<p>

Defines a paragraph of text.

<a>

Creates hyperlinks that connect to other pages or resources.

<img>

Displays images on a webpage using the src attribute.

<div>

A container used to group and organize content for styling and layout.

<ul> and <ol>

Create unordered (bullets) and ordered (numbered) lists.

<table>

Used to create tables with rows and columns for structured data.
