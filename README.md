# Glossary of HTML Tags

A concise and beginner-friendly **HTML Tags Dictionary** built as a quick reference guide for web developers and learners.  
This project provides clear explanations of essential HTML elements, presented with a modern and minimal design.

---

## Overview
Understanding HTML is the foundation of web development. This glossary was created to provide a structured, accessible reference for the most commonly used tags.  
It is designed for students, self-learners, and professionals who want a quick refresher without diving into lengthy documentation.

---

## Features
- Covers fundamental HTML tags such as `<html>`, `<head>`, `<body>`, `<p>`, `<a>`, `<img>`, `<div>`, `<ul>`, `<ol>`, and `<table>`.
- Dark-themed interface for comfortable reading.
- Straightforward definitions for quick learning.
- Organized structure with clear sections for easy navigation.
